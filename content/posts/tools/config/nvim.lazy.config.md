# Nvim.lazy.config

该配置教程根据 https://github.com/xiantang/nvim-conf 进行配置，

## 配置过程
1. 安装 [lazy.nvim](https://github.com/folke/lazy.nvim) 作为插件管理器
2. 更换主题 colorscheme
3. telescope fuzzy finder
4. nvim tree
5. treesitter
6. lsp 配置 golang, lua
    - mason + install lsp
    - lspconfig + lsp server config
    - nvim-cmp + completion
7. git 配置
